Registro da Função

------------------ 

comandos["criar-arquivo"] = [](const std::vector<std::string> &args)
{
    if (args.size() < 1 || args.size() > 3)
    {
        std::cerr << "Erro: Uso incorreto. Uso: criar-arquivo <nome_do_arquivo> [diretorio]" << std::endl;
        return;
    }
    criarArquivo(args[0], args.size() == 3 ? args[2] : "");
};

------------------

Como usar o Comando

Para usar a função de criar arquivo, você pode executar o seguinte comando no terminal:

 criar-arquivo <nome_do_arquivo> [diretorio]

Substitua <nome_do_arquivo> pelo nome do arquivo que você deseja criar e [diretorio] (opcionalmente) pelo diretório onde você deseja criar o arquivo.

Por exemplo:

 criar-arquivo meu_arquivo.txt

Cria um arquivo chamado "meu_arquivo.txt" no diretório atual.

 criar-arquivo meu_arquivo.txt /caminho/para/o/diretorio

Cria um arquivo chamado "meu_arquivo.txt" no diretório especificado.
