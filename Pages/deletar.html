Como usar a função de deletar

CallBack / Functions

------------------  
int main()
{

deletar("meuDiretorio", ".", "-pasta"); // Exclui o diretório chamado "meuDiretorio" no diretório atual
deletar("meuArquivo.txt", ".", "-ar"); // Exclui o arquivo chamado "meuArquivo.txt" no diretório atual
deletar("meuDiretorio", ".", "-all"); // Exclui tudo no diretório chamado "meuDiretorio" no diretório atual

return 0;
}
------------------

Registro da Função

------------------  
comandos["deletar"] = [](const std::vector<std::string> &args)
{
    if (args.size() < 2 || args.size() > 4)
    {
        std::cerr << "Erro: Uso incorreto. Uso: deletar <flag> <nome_do_arquivo_ou_diretorio> [diretorio]" << std::endl;
        return;
    }
    if (args[0] != "-dir" && args[0] != "-all" && args[0] != "-ar" && args[0] != "-pasta") // Adiciona "-pasta" como opção válida
    {
        std::cerr << "Erro: Flag inválida. Use -dir, -all, -ar ou -pasta." << std::endl;
        return;
    }
    deletar(args[1], args.size() == 3 ? args[2] : "", args[0]);
};
------------------

Como usar o Comando

Para usar a função de deletar, você pode executar o seguinte comando no terminal:

deletar <flag> <nome_do_arquivo_ou_diretorio> [diretorio]

Substitua <flag> pela opção desejada (-dir, -all, -ar ou -pasta), <nome_do_arquivo_ou_diretorio> pelo nome do arquivo ou diretório que você deseja excluir e [diretorio] (opcionalmente) pelo diretório onde o arquivo ou diretório está localizado.

Por exemplo:

deletar -ar meuArquivo.txt

Exclui o arquivo chamado "meuArquivo.txt" no diretório atual.

deletar -dir minhaPasta

Exclui o diretório chamado "minhaPasta" no diretório atual.

deletar -all meuDiretorio

Exclui tudo dentro do diretório chamado "meuDiretorio" no diretório atual.
