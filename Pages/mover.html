Como usar a função de mover

CallBack / Functions

------------------ 
int main()
{
    mover("meuDiretorio", "-dir", "/novo/diretorio"); // Move o diretório chamado "meuDiretorio" para "/novo/diretorio"
    mover("meuArquivo.txt", "-ar", "/novo/diretorio"); // Move o arquivo chamado "meuArquivo.txt" para "/novo/diretorio"

    return 0;
}
------------------

Registro da Função

------------------ 
comandos["mover"] = [](const std::vector<std::string> &args)
{
    if (args.size() != 4 || args[2] != "-para")
    {
        std::cerr << "Erro: Uso incorreto. Uso: mover -dir <nome_da_pasta> -para <novo_caminho>" << std::endl;
        return;
    }

    std::string flag = args[0]; // Corrigido para args[0]
    std::string nomePastaOuArquivo = args[1];
    std::string novoCaminho = args[3];

    if (flag == "-dir")
    {
        moverFolder(nomePastaOuArquivo, novoCaminho);
    }
    else if (flag == "-ar")
    {
        moverFile(nomePastaOuArquivo, novoCaminho);
    }
    else
    {
        std::cerr << "Erro: Flag inválida. Use -dir ou -ar." << std::endl;
    }
};
------------------

Como usar o Comando

Para usar a função de mover, você pode executar o seguinte comando no terminal:

mover <flag> <nome_do_arquivo_ou_diretorio> -para <novo_caminho>

Substitua <flag> pela opção desejada (-dir para diretórios, -ar para arquivos), <nome_do_arquivo_ou_diretorio> pelo nome do arquivo ou diretório que você deseja mover e <novo_caminho> pelo caminho para onde deseja mover o arquivo ou diretório.

Por exemplo:

mover -ar meuArquivo.txt -para /novo/diretorio

Move o arquivo chamado "meuArquivo.txt" para o diretório "/novo/diretorio".

mover -dir minhaPasta -para /novo/diretorio

Move o diretório chamado "minhaPasta" para o diretório "/novo/diretorio".

mover -ar meuArquivo.txt -para /novo/diretorio

Move o arquivo chamado "meuArquivo.txt" para o diretório "/novo/diretorio".

mover -dir minhaPasta -para /novo/diretorio

Move o diretório chamado "minhaPasta" para o diretório "/novo/diretorio".
